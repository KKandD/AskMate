Ask mate

☞ Izabela Żmuda, Filip Brzeziński, Paweł Bonecki

Our goal was to implement a crowdsourced Q&A site, like Stack Overflow. The initial version of the site is able to handle questions and answers.

The main funcionalities are: 

        
            → List questions
            → Display a question
            → Ask a question/add an answer
            → Sort questions
            → Add image
            → Vote on question/answer
            → Delete question/answer
            → Edit question

Let's check how it works!!! Take a look at navigation bar 👆.
